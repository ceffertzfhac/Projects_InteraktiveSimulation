Das Vorzeichen von \(\omega\) gibt die Drehrichtung an (negativ = gegen den Uhrzeigersinn); \(|\omega|\) ist die Umlaufgeschwindigkeit (Betrag).
Physik
Gleichförmige ebene Kreisbewegung:
\[\varphi(t) = \varphi_0 + \omega\,t\]
\[x(t) = R\cos(\varphi),\quad y(t) = R\sin(\varphi)\]
\[v_x = -R\,\omega\sin(\varphi),\quad v_y = R\,\omega\cos(\varphi)\]
\[a_x = -R\,\omega^2\cos(\varphi),\quad a_y = -R\,\omega^2\sin(\varphi)\]
Zentripetalbeschleunigung (zum Mittelpunkt gerichtet):
\[|\vec{a}| = R\,\omega^2 = \frac{|\vec{v}|^2}{R},\quad |\vec{v}| = R\,|\omega|\]
Umlaufdauer & Frequenz:
\[T = \frac{2\pi}{|\omega|},\quad f = \frac{1}{T}\]
