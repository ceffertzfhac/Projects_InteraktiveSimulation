Physik
Rollbedingung (ohne Schlupf):
\[v_c = \omega\,R \quad\Rightarrow\quad \omega = \frac{v_c}{R}\]
Schwerpunkt (geradlinig gleichförmig):
\[x_{\text{SP}}(t) = v_c\,t,\quad y_{\text{SP}} = R\]
\[\vec{v}_{\text{SP}} = (v_c,\,0),\quad \vec{a}_{\text{SP}} = \vec{0}\]
Punkt auf dem inneren Kreis \(r = 0{,}9\,R\), Anfangswinkel \(\theta_0\):
\[\varphi(t) = \theta_0 - \omega\,t\]
\[x(t) = v_c\,t + r\cos\varphi(t)\]
\[y(t) = R + r\sin\varphi(t)\]
\[v_x = v_c + r\,\omega\sin\varphi(t)\]
\[v_y = -r\,\omega\cos\varphi(t)\]
\[a_x = -r\,\omega^2\cos\varphi(t)\]
\[a_y = -r\,\omega^2\sin\varphi(t)\]
Punkte: \(\theta_0 \in \{0,\,\tfrac{\pi}{2},\,\pi,\,\tfrac{3\pi}{2}\}\) (P1–P4).
