# Feature Backlog — Kreisbewegung

Ideas and future enhancements, MoSCoW-prioritized.

## Must
(none currently)

## Should
- **Geschwindigkeits-Regler-Richtung**: bei negativer Winkelgeschwindigkeit
  dreht sich die Masse gegen den Uhrzeigersinn — im Live-Panel klarmachen,
  daß |ω| die Umlaufgeschwindigkeit ist und das Vorzeichen die Richtung.
- **Vergleichsbahn / zweite Kreisbahn** mit abweichendem R zum direkten
  Vergleich (analog Schräger-Wurf-Vergleichsbahn).

## Could
- **Umlaufzähler**: wie viele Umläufe absolviert seit Start (φ/360°).
- **Zentripetalkraft-Layer**: \(F_Z = m\,|\vec a|\) als optionale
  Vektorgröße (zusätzliche Masse-Parameter nötig).
- **Winkelgeschwindigkeit auch als rad/s im UI** anzeigen (neben °/s).

## Won't (this version)
- Ungleichförmige Kreisbewegung (variabler ω-Verlauf) — eigener Sim-Typ.
