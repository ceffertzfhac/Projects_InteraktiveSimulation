ILIAS-Probe

Diese Seite testet, was die ILIAS-Umgebung mit einer mehrteiligen (modularen) Web-App macht. Jeder der drei Tests zeigt sein Ergebnis sichtbar auf dieser Seite.

Test 1 — relative CSS-Datei:
Ist diese Box grün umrandet mit hellem Hintergrund? Dann wurde css/style.css über einen relativen Pfad geladen. Fehlt die Umrandung, wird CSS nicht ausgeliefert.
Test 2 — ES-Modul-Import:
Modul-Status: nicht geladen
Steht hier „ES-Modul importiert ✓", wurde eine .js-Datei per import aus einer zweiten Datei nachgeladen (korrekter MIME-Typ, relative Pfade intakt). Bleibt es bei „nicht geladen", erlaubt ILIAS keine ES-Modul-Imports.
Test 3 — MathJax vom CDN:
Formel: \(E = m c^{2}\)
Wird die Formel oben sauber gesetzt (kursive Buchstaben, hochgestelltes 2), ist der CDN im FH-Netz erreichbar. Steht der Roh-Text \(E = m c^{2}\) sichtbar, wird der CDN blockiert.
