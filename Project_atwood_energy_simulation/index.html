Physik
Beschleunigung mit Reibung:
\[a = \frac{(m_1-m_2)\,g - \operatorname{sign}\!\cdot F_R}{m_1+m_2}\]
Haftreibung: \(a=0\) falls \(|(m_1-m_2)\,g| \le F_R\)
Seilkraft:
\[F_S = m_1\,(g - a)\]

\[E_{\text{kin}} = \tfrac{1}{2} m\,v^2,\quad E_{\text{pot}} = m\,g\,(h-h_0)\]
\[E_{\text{ges}} = E_{\text{kin}} + E_{\text{pot}}\]
Reibungsarbeit:
\[E_V = F_R \cdot s\]
Energieerhaltung:
\[E_{\text{ges}} + E_V = \text{konst.}\]
