Physik
Horizontale Bewegung (gleichförmig):
\[x(t) = v_0\cos(\alpha)\,t\]
\[v_x = v_0\cos(\alpha) = \text{const}\]
Vertikale Bewegung:
\[y(t) = h_0 + v_0\sin(\alpha)\,t - \tfrac{1}{2}\,g\,t^2\]
\[v_y(t) = v_0\sin(\alpha) - g\,t\]
\[a_y = -g = -9{,}8\,\tfrac{\text{m}}{\text{s}^2}\]
\[s(t) = v_0\sin(\alpha)\,t - \tfrac{1}{2}\,g\,t^2\]
\[v_y(t) = v_0\sin(\alpha) - g\,t\]
\[a_y = -g = -9{,}8\,\tfrac{\text{m}}{\text{s}^2}\]
\[y(t) = -h_0 - v_0\sin(\alpha)\,t + \tfrac{1}{2}\,g\,t^2\]
\[v_y(t) = -v_0\sin(\alpha) + g\,t\]
\[a_y = +g = +9{,}8\,\tfrac{\text{m}}{\text{s}^2}\]
\[s(t) = -v_0\sin(\alpha)\,t + \tfrac{1}{2}\,g\,t^2\]
\[v_y(t) = -v_0\sin(\alpha) + g\,t\]
\[a_y = +g = +9{,}8\,\tfrac{\text{m}}{\text{s}^2}\]
Flugzeit & Reichweite:
\[t_{\text{fall}} = \frac{v_0\sin(\alpha) + \sqrt{(v_0\sin\alpha)^2 + 2g\,h_0}}{g}\]
\[x_{\text{max}} = v_0\cos(\alpha)\,t_{\text{fall}}\]
\[y_{\text{max}} = h_0 + \frac{(v_0\sin\alpha)^2}{2g}\]
Auftreffwinkel (von der Horizontalen):
\[\alpha_{\text{imp}} = \arctan\!\left(\frac{|v_0\sin(\alpha) - g\,t_{\text{fall}}|}{v_0\cos(\alpha)}\right)\]
