# FH-Physik Simulationen - Share

Dieses Repository enthält verschiedene interaktive Physik-Simulationen und die dazugehörigen Entwicklungsrichtlinien.

## Struktur

- **/global_docs/**: Enthält allgemeine Anleitungen, Architektur-Blueprints (`simulation_instruction.md`) und KI-Kontext-Dateien.
- **/rolling_bodies_simulation/**: Interaktive Simulation zum Rollen von Körpern auf der schiefen Ebene (v1.6.0).
- **/lorentz_force_simulation/**: Interaktive Simulation zur Lorentzkraft zwischen parallelen Leitern (v1.0.0).
- *(Zukünftige Projekte werden hier als weitere Unterordner hinzugefügt)*

## Entwickler-Leitfaden

Wenn Sie eine neue Simulation erstellen möchten, orientieren Sie sich bitte an der [Simulation Instruction](./global_docs/simulation_instruction.md). Diese beschreibt den Standard für Look & Feel, Notation und technische Umsetzung.
